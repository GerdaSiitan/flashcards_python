📚 Flashcards App
🧠 Project Overview

Flashcards App is a simple learning tool inspired by Quizlet.
It allows users to create their own flashcard sets and study them in an interactive way.

The app is designed for:

Students 📖
Self-learners 🎯
Anyone who wants to memorize information efficiently

🎯 Project Goal

The goal of this project is to create an application where users can:

Add their own flashcard topics
Study using a simple question/answer format
Improve memory and learning through repetition
⚙️ Features
📌 Create flashcard topics
📝 Add custom questions and answers
🔁 Flip cards to see answers
🎨 Clean and simple UI
➕ Add more topics easily
